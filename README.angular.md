# Angular Development Notes

Project overview and content-editing notes live in `README.md`. This file only keeps the Angular-specific commands and defaults for the app.

## Development Server

```bash
npm start
```

Default local URL:

```text
http://localhost:4200/
```

## Build

```bash
npm run build
```

Build output is written to `dist/`.

## Tests

```bash
npm test
```

## Angular CLI

Generate new code with:

```bash
npx ng generate component component-name
```

More CLI documentation:

- https://angular.dev/tools/cli
